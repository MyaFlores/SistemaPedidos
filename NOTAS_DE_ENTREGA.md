# Notas de entrega

Esta version combina el Avance 1 y el Avance 2 en un solo programa de consola.

Cambios realizados:

- Se conservaron las clases `Client`, `Product`, `OrderItem` y `Order`.
- Se conservaron los reportes del Avance 2.
- Se reincorporaron las consultas completas del Avance 1 en la salida de consola.
- Se agregaron mas datos de prueba para que los reportes de top, agrupacion y productos no vendidos sean mas claros.
- Se separaron las operaciones principales en funciones propias.
- Se corrigieron textos de salida y errores de escritura.
- Se dejo el proyecto como Kotlin/JVM de consola para que pueda ejecutarse desde Android Studio o terminal.

Evidencia sugerida:

- Ejecutar `./gradlew :app:run`.
- Tomar capturas de pantalla de la consola mostrando las secciones de Entrega 1 y Entrega 2.
- El archivo `evidencia_ejecucion.txt` contiene una ejecucion de referencia.
